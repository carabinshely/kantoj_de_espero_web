# Editorial redesign working notes

Design reference: outer repository PR #10, `DESIGN.md` at `caf91f1`.

## First-listener baseline — exactly three observations

1. **The first viewport reads like a generic catalog landing page.** The old dark gradient and broad “browse songs” choices did not make one confident first listening step obvious. **Acceptance:** the redesigned home leads with a clear listener promise and a primary internal Start Here action.
2. **Playlist discovery is visually flat.** Repeated rounded cards made five curated paths feel interchangeable instead of editorial entry points. **Acceptance:** each playlist uses its existing introduction, ordered tracks, and a distinct data-derived mood treatment.
3. **Lyrics feel like secondary utility content.** The dense dark panel did not invite a first-time listener to read an unfamiliar language. **Acceptance:** song pages give the complete, untouched lyrics a calm, deliberate reading surface after context and listening actions.
